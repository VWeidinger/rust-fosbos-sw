# Die Nadel im Heuhaufen
Bestimme wie oft ein Charakter in einem String ist.

Beispiel

Heuhaufen: **Mein Name ist Jürgen!**
Nadel: **i**
Ergebnis: **2**

Tipp:
```
let my_char = 'C';
let my_string = "Test Wert!".to_string();
```
